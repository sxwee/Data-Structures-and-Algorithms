# 一.什么是链表？

链表是一种线性链式存储结构，其在内存中并**非连续存储**。对于链表，每个数据节点都包含**数据域和指针域**，其中指针域指向当前结点的下一个结点。链表包含很多种形式：**单向或双向，循环的或非循环的**。
