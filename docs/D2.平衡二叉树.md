# 一.平衡二叉树的定义

**结点的高度**：从结点到叶结点的最长路径。

**结点的平衡因子**：指该结点左子树和右子树的高度差。

**平衡二叉树**：各结点的平衡因子的绝对值最大为$1$，即结点的平衡因子只有$-1,0,+1$三种情形。下面便展示了平衡二叉树的例子：
